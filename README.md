# Proof-of-Work Task Manager

This is a backend system built using Django REST Framework where tasks require proof submission instead of just being marked as completed.

## Features
- JWT Authentication
- Task assignment system
- Proof-based submissions
- Task workflow tracking
- Submission review system

## Tech Stack
- Django
- Django REST Framework
- SQLite

## API Endpoints
- /api/token/
- /api/tasks/
- /api/submissions/
